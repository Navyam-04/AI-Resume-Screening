# AI Resume Screening & Talent Analytics Platform

## Overview

A production-grade AI Resume Screening system built using:

- FastAPI
- PostgreSQL
- SQLAlchemy
- LangChain
- Google Gemini
- spaCy
- Streamlit
- Docker
- Redis

## Features

- Resume Parsing
- Skill Extraction
- AI Resume Scoring
- Job Description Matching
- Recruiter Dashboard
- Talent Analytics
